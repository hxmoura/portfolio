O VS Code guarda segredos que poucas pessoas conhecem, uma delas é a função de expor seu projeto publicamente sem precisar hospedar em serviços de cloud (port forwarding).

Esse recurso é útil para mostrar o projeto para outros membros da equipe ou clientes. Por questões de segurança, deve ser usado somente para desenvolvimento, veja como fazer:

1.  Pressione **CTRL + Aspas ( ' )** em seu teclado e acesse a aba "Ports".

    <ImageView imageUrl="/vs01.webp" />
    <Spacer height={36} />

2.  Clique no botão "Forward a port" e informe a porta que está sendo executado seu projeto.

    <ImageView imageUrl="/vs02.webp" />
    <Spacer height={36} />

3.  Você receberá um link de acesso, mas ele ainda é privado (somente você consegue acessar), para permitir o acesso público, clique com o botão direito do mouse sobre **"Private"**. Mude o Port visibility para **"Public"**.
    <ImageView imageUrl="/vs03.webp" />
    <Spacer height={36} />

Pronto! Você terá seu projeto disponível de forma pública e de fácil acesso, basta copiar o link e compartilhar com sua equipe.

**Observações:**

- Ao fazer o processo pela primeira fez, será necessário se autenticar com o Github.
- Para que o link fique disponível, é necessário que o projeto esteja rodando em sua máquina na mesma porta informada no compartilhamento.
